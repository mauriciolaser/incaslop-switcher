# Users

Necesito que tengamos un servicio de usuarios muy básico en el backend. El user debe ser 'inca' y su password 'slop'. Esto debe aparecer como un modal de iniciar sesión antes de cargar la aplicación. En la web se carga con localstorage la sesión para usuarios logeados.

Este servicio es de uso interno, así que no hace fala darle demasiada seguridad, solo necesito ese usuario único.
