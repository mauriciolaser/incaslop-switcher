# Sprint 2

Las muertes están siendo demasiado rápido. Necesito que se vea el último golpe y que luego de que se produzca la muerta haya un delay para que se vea el golpe final y el candidato desapareciendo mientras hace un giro (como caricaturescamente). 

# Descripción del issue
Actualmente, un candidato da el golpe final y se muestra inmediatamente el modal de información de victoria. 

# Comportamiento esperado
El candidato atesta el golpe final. El candidato que pierde inicia un ciclo de "derrota". Luego de que acaba el ciclo se muestra el mensaje del personaje ganador. 

# Animación
El personaje perdedor desaparece con una animación ad hoc. Tu tarea es generar la animación así como has generado el resto de sprites del personaje, con el mismo estilo.

# Estilo
Añádale dramatismo y efectos, distorsiones de pantalla, etc. al producirse la muerte de uno de los luchadores.
