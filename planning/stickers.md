# Stickers

Vamos a crear un sistema de stickers usando gifs. 

En el dashboard se añade un campo GIF que tiene para completar con un link a un archivo gif público. Este se debe renderizar en la página y existir hasta que se acaba y aparecer en una posición aleatoria.

## Protección

Se debe proteger de la inyección de archivos no compatibles. Deben ser archivos .gif.
